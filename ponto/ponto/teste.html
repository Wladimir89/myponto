Java SE

Ferramentas essenciais para programação em Java, incluindo um editor, depurador, Profiler, suporte à refatoração e uma ferramenta de desenvolvimento visual arrastar-e-soltar (anteriormente conhecida como Projeto Matisse).

JavaFX

Java Web e EE

Ferramentas para criação de aplicações Java web e enterprise compatíveis com especificações J2EE 1.4 e Java EE 5. Inclui suporte para servlets, JSPs, bibliotecas de balas, Spring, Struts, API Java Persistence, Enterprise Java Beans, JAX-WS e serviços Web RESTful e Java Server Faces, a linguagem Groovy e o framework Grails. Fornece um ambiente visual de desenvolvimento de aplicações arrastar-e-soltar.

Java ME

Ferramenta para criação de aplicações Java Micro Edition para telefone celulares e dispositivos portáteis. Inclue desenvolvimento visual, Assistente de conexão de serviços web, e ferramentas para manipulação de problemas de fragmentação de dispositivo.
Java ME
Ruby

Ferramenta completa para a criação de aplicativos Ruby e Ruby on Rail. inclue interpretador JRuby, framework Ruby on Rails, super editor, depurador, gerenciador, e interatividade do Ruby com shell. Suporta fácil criação, modificação, e execução de aplicativos Ruby on Rails.

C/C++
